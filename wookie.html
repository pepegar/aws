wookie 

wookie is a library for using Amazon Web Services services from Scala.

add to your project 

"io.pepegar.gihub" %% "wookie-core" % "compile"

Each one of the AWS Services are in its own isolated project, and they follow the following namging structure: 

"io.pepegar.gihub" %% "wookie-{service}" % "compile"

So, for example, if you want to use DynamoDB, 

"io.pepegar.gihub" %% "wookie-dynamodb" % "compile"

Currently implemented services 

Currently we have implementation for:

  • DynamoDB
  • S3
